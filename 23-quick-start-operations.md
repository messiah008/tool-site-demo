## 十、止损决策清单

### 10.1 阶段性止损点

**Week 1 末(首周 UV)**:
- [ ] 检查 UV ≥ 100 → 继续
- [ ] UV < 100 → 排查 SEO/域名/收录

**Week 4 末(月 UV)**:
- [ ] 检查月 UV ≥ 500 → 继续
- [ ] 月 UV 100-500 → 优化内容
- [ ] 月 UV < 100 → **停止扩工具,排查根本问题**

**Week 6 末(数据库工具后)**:
- [ ] 检查月 UV ≥ 2000 → 继续
- [ ] 月 UV 500-2000 → 优化 SEO
- [ ] 月 UV < 500 → **转为技术作品,停止投入**

**Week 10 末(付费工具后)**:
- [ ] 检查月付费订单 ≥ 10 → 继续
- [ ] 月付费 1-9 → 优化转化
- [ ] 月付费 0 → **放弃付费线,只做免费引流**

**Week 12 末(最终门禁)**:
- [ ] 月收入 ≥ 300 元 → 持续运营
- [ ] 月收入 100-300 → 兼职维持
- [ ] 月收入 < 100 → **转为技术作品,停止投入**

### 10.2 立即停止信号

| 信号 | 行动 |
|------|------|
| 域名被百度惩罚 | 换域名重启 |
| WPS License > 10000 元/年 | 放弃 WPS 路线,用 PaddleOCR |
| 月固定成本 > 300 元且月收入 < 500 元持续 2 月 | 暂停投入 |
| 闲鱼账号被封 | 转私域 + 多账号 |
| 备案被拒 | 转海外服务器 + 海外流量 |
